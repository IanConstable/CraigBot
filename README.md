# CraigBot
Simple Discord bot that posts tweets from a specified account in real time, designed for @CraigWeekend


## appsettings.json
```
{
    "Settings":
    {
        "BotSettings":
        {
            "ServerID": ,
            "ChannelID": ,
            
            "TwitterID": ,
            "TwitterScreenName": ""
        },

        "TwitterKeys":
        {
            "ConsumerKey": "",
            "ConsumerSecret": "",
            "AccessToken": "",
            "AccessTokenSecret": ""
        },

        "DiscordToken": ""
    }
}
```
